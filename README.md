# Chloe Recupero — Personal Portfolio

<img src="/cover.jpg" alt="Portfolio preview" width="100%" />

A minimalist, animated personal portfolio built with **Next.js 15**, **React 19**, **Tailwind CSS**, and **Framer Motion**.

This site showcases my work, experience, and projects in a clean, user-focused way — with an emphasis on thoughtful design, smooth interactions, and real-world impact.

## ✨ Features

- Clean, minimal portfolio layout
- Interactive **project carousel** with smooth transitions
- Dedicated **Projects, Experience, About, and Contact** pages
- Responsive design across devices
- Subtle animations using Framer Motion
- Custom typography (including handwritten hero accent)
- Resume integration + external links

---

## 🧠 Design Goals

This project focuses on:
- **Clarity over clutter** — simple layouts that highlight content
- **Human-centered design** — approachable, personal tone
- **Real-world storytelling** — not just what I built, but why it matters
- **Subtle motion** — animations that enhance, not distract

---

## 🛠 Tech Stack

- **Next.js 15 (App Router)**
- **React 19**
- **Tailwind CSS**
- **Framer Motion**
- **next-themes** (dark mode support)
- **next/font** (custom typography)
